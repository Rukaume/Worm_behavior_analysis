# Worm_behavior_analysis
====

This script is for behavior analysis with pillar chamber.   

Input data  
1. recorded images (.jpg or .png)
2. ROI (.csv)
Made by ImageJ

## Usage  
1. run the script  
2. select images 
3. select ROI csv 
4. wait for calculation
